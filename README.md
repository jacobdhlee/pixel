# Pixel

Photos are from Pixels API. Users can browse photos or search

## Getting Started

First npm install

```bash
    # go to project
    cd pixel

    # npm install
    npm install
```

Second add evn file

```bash
    #create file .env.local
    touch .env.local
    #in .env.local
    API_KEY=YOUR PIXEL API KEY HERE

```

Three, run the development server:

```bash
    npm run dev
```
